## Responsive Behavior

### Desktop (lg breakpoint)
- Two-column layout for Tailor Resume module (keywords + bullets side-by-side)
- Header actions in single row
- Status dropdown on right side

### Tablet (md breakpoint)
- Single column layout for some sections
- Buttons might stack

### Mobile (sm breakpoint)
- Full-width buttons (flex-1 or w-full)
- Single column layouts
- Status dropdown repositioned
- Collapsible sections expand full width
